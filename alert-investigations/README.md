# Alert Investigations

This folder contains compact, case-based alert investigations. Each case is written to show analyst reasoning: how the alert was triaged, which telemetry was used, what evidence supported the decision, and what response actions were justified.

## Cases

| Case | Title | Focus |
|---|---|---|
| 001 | [Remote Code Execution against Splunk Enterprise](./001-splunk-rce/README.md) | RCE validation, endpoint correlation, persistence evidence |
| 002 | [Famous Chollima Fake-Recruitment Compromise](./002-chollima-endpoint-forensics/README.md) | PowerShell delivery, payload staging, Run-key persistence |
| 003 | [RedLine Memory Forensics Investigation](./003-redline-memory-forensics/README.md) | Volatility process analysis, memory permissions, network indicators |
| 004 | [RevengeHotels Multi-Stage Endpoint Compromise](./004-revengehotels-apt/README.md) | Script execution, defense impairment, persistence, collection staging |
